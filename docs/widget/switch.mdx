title: NakedSwitch
---

NakedSwitch provides on/off switch behavior and accessibility without imposing any visual styling. You control all visuals via `child` or `builder`, while the widget handles focus, keyboard activation, hover/press callbacks, semantics, and toggling.

<Info>
  Example: https://github.com/btwld/naked_ui/blob/main/example/lib/api/naked_switch.0.dart
</Info>

<CodeGroup title="Usage Example" defaultLanguage="dart">
```dart
class SwitchExample extends StatefulWidget {
  const SwitchExample({super.key});
  @override
  State<SwitchExample> createState() => _SwitchExampleState();
}

class _SwitchExampleState extends State<SwitchExample> {
  bool _value = false;
  bool _hovered = false;
  bool _pressed = false;
  bool _focused = false;

  Color get _bg {
    if (_pressed) return const Color(0xFF3D3D3D).withValues(alpha: 0.9);
    if (_hovered) return const Color(0xFF3D3D3D).withValues(alpha: 0.8);
    return const Color(0xFF3D3D3D).withValues(alpha: 0.7);
  }

  @override
  Widget build(BuildContext context) {
    return NakedSwitch(
      value: _value,
      onChanged: (next) => setState(() => _value = next ?? false),
      onHoverChange: (h) => setState(() => _hovered = h),
      onPressChange: (p) => setState(() => _pressed = p),
      onFocusChange: (f) => setState(() => _focused = f),
      child: Container(
        width: 44,
        height: 24,
        decoration: BoxDecoration(
          color: _value ? _bg : const Color(0xFFE5E7EB),
          borderRadius: BorderRadius.circular(12),
          border: Border.all(
            color: _focused ? const Color(0xFF111827) : const Color(0xFF00000000),
            width: 2,
          ),
        ),
        alignment: _value ? Alignment.centerRight : Alignment.centerLeft,
        padding: const EdgeInsets.all(2),
        child: Container(
          width: 20,
          height: 20,
          decoration: const BoxDecoration(
            color: Colors.white,
            shape: BoxShape.circle,
          ),
        ),
      ),
    );
  }
}
```
</CodeGroup>

## Constructor

```dart
const NakedSwitch({
  Key? key,
  this.child,
  required this.value,
  this.onChanged,
  this.enabled = true,
  this.mouseCursor,
  this.enableFeedback = true,
  this.focusNode,
  this.autofocus = false,
  this.onFocusChange,
  this.onHoverChange,
  this.onPressChange,
  this.builder,
  this.semanticLabel,
})
```

## Properties

- child: Visual content for the switch. Provide either `child` or `builder`.
- value: Current on/off state (bool). Must be nonâ€‘null.
- onChanged: Called with next value on toggle. If null, switch is disabled.
- enabled: Whether the switch is interactive.
- mouseCursor: Cursor to show when hovered.
- enableFeedback: Platform feedback on activation.
- focusNode/autofocus: Focus management.
- onFocusChange/onHoverChange/onPressChange: State callbacks.
- builder: Receives `Set<WidgetState>` for visuals.
- semanticLabel: Optional accessibility label.

## Accessibility

- Exposes proper toggle semantics (`toggled`, `enabled`).
- Keyboard activation via Enter/Space.
- Merges semantics with focus wrapper for a single, clear node.
